Custom Shield Generator

Generate your own shield by adding parameters to the URL:

  • label: The left text (e.g., ?label=Build)
  • message: The right text (e.g., &message=Passing)
  • labelColor: Hex color for the label background (e.g., &labelColor=%23ff5733)
  • messageColor: Hex color for the message background (e.g., &messageColor=%234c1)

Example URL: ?label=Build&message=Passing&labelColor=%23ff5733&messageColor=%234c1
